# Hospital-management-system--


A console-based Hospital Management System built with **C++** that allows 
users to manage patients, doctors, and appointments efficiently.

---

## 🚀 Features

- ➕ Add, View, Search, and Delete **Patients**
- 🩺 Add and View **Doctors**
- 📅 Book and View **Appointments**
- 🔒 Simple and clean menu-driven interface

---

## 🛠️ Built With

- **Language:** C++
- **Concepts:** Arrays, Functions, Loops, OOP
